# 🚀 UFO Galaxy 极简上手指南

> **一句话介绍**：这是一个全自主的 AI 智能体系统，能帮你操作电脑、整理文件、分析数据，甚至自己写代码。

---

## 第一步：一键启动 (操作)

**在 Linux/Mac 终端中：**
```bash
./start.sh
```

**在 Windows 中：**
双击 `start.bat`

*系统会自动检查环境、安装依赖，并启动所有服务。首次运行会进入配置向导，按回车使用默认值即可。*

---

## 第二步：打开控制台 (交互)

启动成功后，浏览器访问：
👉 **http://localhost:8080**

你会看到一个简洁的 **API Manager** 界面：
1.  **Dashboard**: 查看系统健康状态（CPU、内存、节点在线情况）。
2.  **Chat**: 直接与 AI 对话（例如：“帮我整理下载文件夹里的 PDF 文件”）。
3.  **Settings**: 配置你的 API Key（OpenAI, Gemini 等）。

---

## 第三步：高效交互 (使用)

系统专为**键盘流**用户设计，支持“硬件唤醒 -> 文本交互”的高效流程：

1.  **一键唤醒**：按下全局快捷键 `Ctrl+Shift+Space`（默认）。
2.  **即刻输入**：系统会立即弹出输入框并自动聚焦，**无需鼠标点击**。
3.  **指令直达**：直接输入指令（如 "整理桌面"），回车执行。
4.  **自动隐藏**：任务完成后，界面会自动收起，不打扰你的工作。

> **注意**：默认关闭了语音监听以保护隐私。如需开启，请在 `config.json` 中设置 `"enable_voice": true`。

---

## 常见问题

**Q: 怎么停止？**
A: 在终端按 `Ctrl+C`，系统会优雅地关闭所有节点。

**Q: 节点显示红色（离线）怎么办？**
A: 检查 `logs/` 目录下的日志，或者在控制台点击“重启节点”。

**Q: 我需要一直开着终端吗？**
A: 是的。如果你想让它在后台完全静默运行，可以使用 `nohup ./start.sh &` (Linux/Mac)。
